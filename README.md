## Getting Started:

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features:

- Login with your web3 wallet
- View NFT collection

## Testing:
- All the above completed features were tested via MetaMask
